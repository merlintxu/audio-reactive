# Contributing

Gracias por tu interés en contribuir.

Guía rápida:

- Reporta issues en GitHub con pasos para reproducir.
- Para cambios pequeños: crea una rama, haz commit y abre un PR apuntando al `master`.
- Ejecuta `npm ci` y revisa el lint con `npx eslint .` antes de abrir PRs.

Para desarrolladores:

1. Instala dependencias: `npm ci`
2. Inicia la app: `npm start`
3. Abre PRs pequeños y enfocados; referencia issues relacionados.
